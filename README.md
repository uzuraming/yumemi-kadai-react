# yumemi-kadai-react

ゆめみさんのフロントエンド課題として作成した課題です。

# デモ

https://fervent-raman-636b45.netlify.app/

# 環境

- Node.js 14.15.0
- TypeScript 4.4.2
- Next.js 11.1.2
- Axios 0.21.4
- React 17.0.2
- Recharts:2.1.4

# 開発環境構築

```bash
git clone https://github.com/uzuraming/yumemi-kadai-react.git
cd yumemi-kadai-react
npm install
npm run start
```

# ファイル構造

| ディレクトリ | 機能                    |
| ------------ | ----------------------- |
| components   | コンポーネント          |
| helper       | 便利な関数を入れる場所  |
| pages/api    | API との接続部分        |
| pages        | アプリの表示部分        |
| styles       | css                     |
| test         | かんたんな API のテスト |
| types        | 型の定義                |
